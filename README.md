# 🎮 Number Guessing Game

A dynamic, interactive command-line application built with Python. The game challenges players to guess a randomly generated hidden number within multiple predefined ranges while maintaining local score tracking.

## 🚀 Features

- **Dynamic Difficulty Tiers**: Three tailored range settings to choose from:
  - **Easy**: Ranges from 1 to 10
  - **Medium**: Ranges from 1 to 50
  - **Hard**: Ranges from 1 to 100
- **Interactive Prompts**: Real-time hot/cold direction updates ("Too High" or "Too Low") to intelligently guide the user.
- **Session Scoreboards**: Local persistence tracking for overall attempt quantities and high score updates.

---


## 📦 Getting Started

### 1. Run the Game
Execute the application directly inside your console using:
```bash
python number_guessing_game.py
```

## 🎮 How To Play

1. **Select Difficulty**: Enter `1`, `2`, or `3` into the prompt area to configure your grid matrix.
2. **Submit Input**: Guess your value and monitor feedback.
3. **Beat Your Record**: Complete the session in as few moves as possible to lower your benchmark tracking record!
